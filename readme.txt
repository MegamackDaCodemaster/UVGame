Quest of the Fallen Depths Readme


To run the game, follow these steps:

1. Run the script 'run-game.sh'. (Do this by opening a terminal within the root folder of this project (UVGame/) and enter 'bash run-game.sh')

2. Select "Play" in the menu!


If the above does not work try these steps instead:

1. Open a terminal within the root folder of this project. (UVGame/)

2. Type 'mvn package' and enter. Wait until all tests have completed and you see the 'Build Success' message.

3. Type 'java -jar target/UVGame-1.0-SNAPSHOT.jar' and enter.

This should launch the main menu of the game. Select 1 to start playing. Enjoy!
